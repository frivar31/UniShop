Tâche 1 : Analyse

Maintenant que vous êtes familier avec le modèle fonctionnel du projet UniShop, vous devez estimer ce qui est nécessaire pour mettre le système en opération. Dans un paragraphe de 100 mots (+/-10%), estimez les informations suivantes :

-Le programme UniShop inclura les fonctionnalités suivantes :

-la gestion des profils d'acheteurs et de revendeurs, le répertoire de produits, les processus d'achat, de règlement, de suivi des commandes, de traitement des retours, de communication entre acheteurs et revendeurs, ainsi que la collecte de données.
Les données seront stockées dans une base de données centralisée, sécurisée et hébergée dans un cloud ou un centre de données.
Concernant l'équipement nécessaire, un serveur pour l'application web, des dispositifs pour les utilisateurs (ordinateurs, smartphones) et une infrastructure réseau pour rependre aux besoins.
Le système interagira avec d'autres systèmes de paiement, des systèmes de suivi des commandes, et éventuellement des services pour les notifications.
Les utilisateurs interagiront avec le système via une interface web et des applications mobiles pour accéder à toutes les fonctionnalités.
