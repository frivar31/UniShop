Tâche 1 : Analyse

Maintenant que vous êtes familier avec le modèle fonctionnel du projet UniShop, vous devez estimer ce qui est nécessaire pour mettre le système en opération. Dans un paragraphe de 100 mots (+/-10%), estimez les informations suivantes :

-UniShop inclura les fonctionnalités suivantes :

-On informatisera les parties liées à l'inscription, la gestion des profils, le catalogue de produits, la recherche, le panier, les commandes, les évaluations, le suivi d'acheteurs, les promotions et les notifications.
-Les données seront stockées dans une base de données sécurisée sur un serveur cloud pour garantir disponibilité et gestion efficace.
-Le matériel requis inclura des serveurs pour l'hébergement, une infrastructure réseau sécurisée et des dispositifs pour les utilisateurs.
-UniShop interagira avec d'autres systèmes via des API pour les paiements, expéditions, et notifications.
-Les utilisateurs interagiront via une ligne de commande, permettant l'inscription, la recherche, l'achat, l'évaluation, et la gestion de profils.
